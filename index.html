Sincronización entre dispositivos opcional, usando una base Supabase

Por defecto todos los datos quedan guardados solo en este navegador. Si querés ver lo mismo desde el celular y la computadora, podés conectar una base de Supabase gratuita. Es opcional: si no la configurás, la app sigue funcionando igual que ahora, solo local.

Sin configurar todavía
"Subir" reemplaza lo que hay en el servidor con lo que ves en este dispositivo. "Traer" hace lo opuesto: reemplaza lo de este dispositivo con lo del servidor. Usalos con cuidado si estás sincronizando desde dos lugares a la vez.

Cómo crear la tabla en Supabase pegar una sola vez en el SQL Editor de tu proyecto

create table deposito_sync ( id text primary key, data jsonb not null, updated_at timestamptz default now() ); alter table deposito_sync enable row level security; create policy "acceso por codigo de espacio" on deposito_sync for all using (true) with check (true);

Esta política deja leer y escribir a cualquiera que tenga la URL, la clave anon y el código de espacio exactos — como una contraseña compartida. Sirve para uso personal entre tus propios dispositivos; no compartas esos tres datos.

Publicar como app instalable GitHub Pages

Subí los archivos index.html, manifest.json, sw.js y los íconos a un repositorio de GitHub, activá GitHub Pages (Settings → Pages → rama main, carpeta raíz) y vas a tener una URL https://tu-usuario.github.io/tu-repo/. Al entrar desde el celular, el navegador va a ofrecer "Agregar a pantalla de inicio" / "Instalar app" — o podés tocar el botón Instalar app de la barra lateral.

La instalación necesita que el sitio esté en HTTPS (GitHub Pages ya lo es) — abrir el archivo suelto con doble clic no permite instalarlo como app, aunque la app funcione igual.
